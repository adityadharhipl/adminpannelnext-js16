
## Installation

Follow the steps below to set up and run the project locally.

### 1. Install Dependencies

After cloning or downloading the repository, navigate to the project directory and install the required dependencies.

Using **npm**:

```bash
npm install
```

Using **Yarn**:

```bash
yarn install
```

Using **pnpm**:

```bash
pnpm install
```

---

### 2. Start the Development Server

Once the dependencies are installed, start the development server.

Using **npm**:

```bash
npm run dev
```

Using **Yarn**:

```bash
yarn dev
```

Using **pnpm**:

```bash
pnpm dev
```

---

### 3. Open in Browser

Open your browser and visit:

```
http://localhost:3000
```
